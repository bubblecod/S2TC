# Subnautica 2 非官方繁體中文

## 翻譯重點
1. 提取官方原始簡體中文翻譯，透過繁化姬轉換為台灣繁中，再手動微調細節<br>
_窗口 -> 視窗_<br>
2. 若存在於Subnautica前作中相同之物件，則依照其官方命名修改<br>
_裝配台 -> 加工機台_<br>
3. 修改人名由義譯改為音譯，避免在長語句中混淆，不易閱讀<br>
_幸福 -> 菲利克斯_<br>
4. 調整不通順及易混淆語句及語意，或參照官方日、英翻譯作調整<br>
_禁用/啟用 -> 已停用/啟用中_<br>
5. 修改字型、美化標點符號(Noto Sans Simplified Chinese -> Noto Sans Traditional Chinese)<br><br>


## 限制
**1. 翻譯不會包含開始畫面右側的遊戲公告，此處還有可能會有缺字問題**<br>
**2. 此翻譯目前僅在Steam平台測試，其他平台不保證能使用**<br>
**3. 此為非官方翻譯，不保證所有文本的翻譯品質，亦不保證隨時與官方更新時程一致及更新後的可用性**<br><br>

## 安裝方法

Windows使用者推薦下載exe執行檔，可自動安裝或輕鬆解除安裝此翻譯。手動可下載ZIP壓縮包，解壓縮後會包含pak、ucas、utoc三個檔案，將其放入遊戲Subnautica2-Windows.pak相同資料夾中(通常是```...Subnautica2\Content\Paks\```)即完成。**安裝成功後，需在遊戲語言選項中選擇中文(简体)並套用。**
